# Contribuindo

Contribuições são bem-vindas.

## Diretrizes

- Não envie credenciais reais.
- Não envie nomes de empresas, domínios internos, IPs internos ou hostnames reais.
- Use exemplos genéricos como `smtp.exemplo.com`.
- Valide a sintaxe dos scripts antes de enviar alterações.

```bash
bash -n instalador/*.sh
bash -n testes/*.sh
```
